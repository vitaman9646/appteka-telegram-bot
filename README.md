# 🤖 Appteka Telegram Bot

Автоматический парсер новых приложений из [Appteka.store](https://appteka.store) с публикацией в Telegram-каналы.

## 🎯 Возможности

- ✅ Автоматический парсинг новых приложений каждые 30 минут
- ✅ Фильтрация по категориям (игры, инструменты, премиум и т.д.)
- ✅ Умная фильтрация дублей (не постит одно и то же дважды)
- ✅ Поддержка нескольких каналов одновременно
- ✅ Красивое форматирование постов с иконками
- ✅ База данных для отслеживания опубликованных приложений
- ✅ Логирование всех операций

## 📋 Требования

- **VPS/сервер** (Ubuntu 20.04+, Debian 11+, или любой Linux)
- **Python 3.8+**
- **Telegram боты** (создаются через @BotFather)
- **Telegram каналы** (боты должны быть админами)
